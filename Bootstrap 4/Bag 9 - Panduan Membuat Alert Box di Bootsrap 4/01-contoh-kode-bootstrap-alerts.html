Alerts

Dengan menggunakan Bootstrap 4, kita bisa membuat Alert Message dengan mudah. Berikut adalah contoh tampilan Alert yang bisa kita buat di Bootstrap 4:

Success! Alert Box ini menandakan tindakan positif atau tindakan yang berhasil.
Info! Alert Box ini menandakan suatu informasi.
Warning! Alert Box ini menandakan Warning (Peringatan) atau tindakan yang perlu diperhatikan.
Danger! Alert Box ini menandakan suatu peringatan bahaya atau tindakan yang negatif.
Primary! Alert Box ini menandakan tindakan yang penting.
Secondary! Alert Box ini menandakan tindakan yang dianggap biasa saja.
Dark! Alert Box abu-abu gelap.
Light! Alert Box abu-abu terang.
