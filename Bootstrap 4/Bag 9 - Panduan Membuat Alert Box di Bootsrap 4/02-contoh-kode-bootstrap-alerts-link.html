Alerts Link

Jika kalian ingin membuat Link yang berada di dalam Alert Box, kalian dapat menggunakan class .alert-link. Berikut adalah contohnya:

Success! Kalian dapat membacanya di Link ini.
Info! Kalian dapat membacanya di Link ini.
Warning! Kalian dapat membacanya di Link ini.
Danger! Kalian dapat membacanya di Link ini.
Primary! Kalian dapat membacanya di Link ini.
Secondary! Kalian dapat membacanya di Link ini.
Dark! Kalian dapat membacanya di Link ini.
Light! Kalian dapat membacanya di Link ini.
